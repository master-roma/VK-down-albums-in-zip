# VK-down-albums-in-zip
[En] VK down albums in .zip
In each album, it adds a "Download album to .zip" button, which downloads all the photos into a single zip file. Everything is simple and convenient.

[Ru] VK качалка альбомов в .zip
В каждом альбоме добавляет кнопку "Скачать альбом в .zip", при нажатии на которую скачиваются все фотографии в один zip файл. Всё просто и удобно.
